  -> A movement sensor detects when the cat is less than 10cm close to the box and triggers the servomotr which controls the box door to open
  -> While the servomotor openes the door, a green led is flashing
  -> After the door is closed, a red led is flashing
  
  -> There is a button connected to a servomotor which controls the cover of the box (to let the owner put food in the box)
  -> When the button is pressed, the cover opens and when pressed again, the cover closes.
